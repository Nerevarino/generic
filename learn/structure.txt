Конечная структура сайта:

Корневая папка:
  папка страницы1
  папка страницы2
  ...............
  ...............
  ...............
  папка страницыN
  img - папка, содержая иконки и изображения
  video - папка, содержащая видеофайлы
  gif - папка, содержащая анимированные картинки

Папка страницыI:
  page.php - главный файл, именно к нему приходит обращение, в нем содержится все необходимое для обработки сервером
  page.css - файл всех стилей, необходимых для страницы
  page.js  - файл всех клиентских скриптов, необходимых для страницы


По приходу клиентского запроса сервер проверяет, базовый ли это запрос страницы (PageRead),
запрос ли это каких-то отдельных данных (Ajax запрос DataRead)  или это ввод данных на сервер (DataWrite)

PageGet: если обычный GET или $_POST запрос без входных данных, всегда возвращает страницу сайта
ReadData: если $_POST запрос с входными данными и с типом "r", то скрипт обрабатывает данные, формирует выходные и отправляет клиенту командой echo
          также строгое соответствие с типовыми запросами, если запрос не типовой, то он игнорирутеся/отправляется ответ о мошенничестве
WriteData: $_POST запрос по строгой форме с проверкой прав доступа на запись, внесение изменений в данные или не внесение с отбивкой пользователю об успехе/провале
